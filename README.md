## Bootstrapped with

- [Reactjs](https://reactjs.org/) - frontend library
- [Jest](https://jestjs.io/) - testing

## Getting started

1. You need node installed globally on your local machine

```
$ git clone https://github.com/blossom-babs/keeper-app
$ cd keeper-app
$ npm install
$ npm run build
```
